GETTING FILES
	Choice 1: from Email attachment
		1) Download attachment from email submission

	Choice 2: from Git Repository
		Install Git 2.13.0 
			1) Download Git-2.13.0-64-bit.exe from https://git-scm.com/download/win
			2) Open Git-2.13.0-64-bit.exe and follow the prompts to install Git.

		Clone Git Repository
			1) Open a folder where you want to clone the git repository.
			2) Right-click and press "Git Bash here"
			3) Enter "git clone https://github.com/YsabelDoquenia/CS153Project.git"
	
RUNNING THE PROGRAM
	Choice 1: Using Python
		Install Python 2.7.13
			1) Download python-2.7.13.msi from https://www.python.org/downloads/release/python-2713/
			2) Open python-2.7.13.msi and follow the prompts to install Git.
			3) Make sure to include in your Environmental Variable's Path the location of your Python27 folder ("C:\Python27").

		Compile Python File	
			1) Open the folder containing GaloisCalculator.py.
			2) In the address bar of Windows Explorer, type "cmd" and enter.
			3) Enter "python GaloisCalculator.py".
			4) Enjoy using the Calculator!
